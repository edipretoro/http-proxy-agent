h1. HTTP-Proxy-Agent

h2. INSTALLATION

To install this module, run the following commands:

bq.	perl Makefile.PL

bq.	make

bq.	make test

bq.	make install

Alternatively, to install with Module::Build, you can use the following commands:

bq.	perl Build.PL
	./Build
	./Build test
	./Build install

h2. SUPPORT AND DOCUMENTATION
